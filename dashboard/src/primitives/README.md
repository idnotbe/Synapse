# Dashboard Primitives

Command Center views import layout and disclosure primitives from `dashboard/src/primitives`.

- `AppShell`, `PageHeader`, and `Section` define the shared frame and required tier/questions metadata.
- `StatCard`, `MetricRow`, `DataTable`, `FleetRow`, and `AgentPeek` keep overview, triage, and drill-down layouts consistent.
- `ToolCallCard`, `TranscriptTurn`, and `RawValue` keep hostile or verbose content summarized first, with raw verification behind a collapsed disclosure.

Use the design tokens in `dashboard/design` for spacing, color, type, focus, and density behavior.
